# Aivonx Proxy 文件

## 快速導覽

- **入門**：介紹、需求與部署指南
- **參考**：設定、API 端點與 CLI 指令
- **資源**：故障排除、安全最佳實作與術語表
- **開發**：程式碼結構、貢獻指南與測試

## 在本機啟動文件伺服器

在開發環境中可使用 MkDocs 啟動預覽：

```bash
mkdocs serve -f mkdocs.yml
```

欲參考原始程式碼，請查看 `src/` 下的模組（例如 `src/aivonx/settings.py`、`src/proxy/`）。
