# 簡介

Aivonx Proxy 是一個基於 Django 的反向代理與管理服務，設計用於 Ollama 模型伺服節點。

## 主要功能

- **節點管理**：管理多個模型服務節點，支援 active/standby 池的分配
- **REST API**：提供節點管理、模型拉取、狀態監控與健康檢查等端點
- **Web UI**：內建管理介面（位於 `ui/` 與 `proxy.web`）供登入與管理使用

## 核心元件

- **程式碼根目錄**：`src/`
- **專案設定**：`src/aivonx/settings.py`
- **Proxy 邏輯**：`src/proxy/`

## 架構概覽

代理扮演統一閘道，會依據以下條件將請求路由到後端 Ollama 節點：
- 模型是否存在於節點上
- 節點健康狀態
- 負載平衡策略（最少活動或最低延遲）
- 即時監控與故障轉移機制
